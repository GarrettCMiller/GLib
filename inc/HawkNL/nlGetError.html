nlGetError, nlGetSystemError

get an error message

C Specification

NLenum nlGetError(void)

NLint nlGetSystemError(void)

Parameters

none

Description

nlGetError and nlGetSystemError retrieve that last NL or system error. The error is not reset, but it can be changed by subsequent NL socket calls. The error can be converted to a string with nlGetErrorStr or nlGetSystemErrorStr.

Return values

The last error.

Error codes

none
