nlInit, nlShutdown

initialize or shutdown NL

C Specification

NLboolean nlInit(void)

void nlShutdown(void)

Parameters

none

Description

nlInit initializes NL, and nlShutdown shuts down NL.

Return values

If nlInit succeeds, then it returns NL_TRUE, otherwise it returns NL_FALSE and an error code is set.

Error codes

NL_NO_NETWORK : No network provider could be intitialized.
NL_OUT_OF_MEMORY : Not enough memory.
